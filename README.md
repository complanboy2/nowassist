# NowAssist

Professional Chrome extension toolkit for ServiceNow developers and engineers working with authentication, APIs, and debugging.

## Features

- **JWT Decoder**: Decode, verify, and analyze JSON Web Tokens
- **JWT Encoder**: Create and encode custom JWTs
- **SAML Inspector**: Inspect and analyze SAML assertions
- **REST API Tester**: Test REST APIs with custom headers and authentication
- **HAR Analyzer**: Analyze HTTP Archive files
- **JSON Utility**: Format, validate, and manipulate JSON data
- **Encoder/Decoder**: Encode and decode using multiple formats

## Privacy

Everything happens in your browser. Your data never leaves your device.
